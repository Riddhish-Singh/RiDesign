Overview

Blocks are the building units: hero, text, image, video, two-col, button, card, section. Each block has editable properties in the left editor and renders instantly in the live preview on the right.

Quick workflow (summary)
  1. Open RiDesign.html (builder).
  2. Select a theme from the Theme dropdown.
  3. Click a block button (Hero / Text / Image / Video / 2-Col / Button / Card / Section) to add it.
  4. Edit the block in the left editor (text fields, file upload, animation).
  5. Drag blocks in Blocks (drag to reorder) to change order.
  6. Use Download .html, Copy HTML or Share Site to export.

Card & Section blocks

Cards are boxed UI elements with title + body. Sections allow background color blocks to visually separate page areas.

// Example: add a section with background and then a card inside via editor
addBlock('section');
const section = state.blocks[state.blocks.length-1];
updateBlock(section.id, { title: 'Pricing', bg_color: '#f7f7fb' });
addBlock('card'); // drag a card below the section or reorder
renderPreview();
Reorder, Duplicate, Remove

Drag items in Blocks (drag to reorder) to change position. Use the duplicate button to clone a block. Use Remove in the editor to delete.

Animations

Animation controls appear in block editor (type, duration in ms, delay). Keep durations short (300â€“900ms) for crisp UX.

Export & Share
  1. Click Download .html to save a single-file static export (images/videos embedded).
  2. Click Copy HTML to paste into another editor or CMS.
  3. Click Share Site to create a temporary blob link for quick sharing.

Reminder: exported HTML contains embedded images/videos when files were uploaded through the builder.
